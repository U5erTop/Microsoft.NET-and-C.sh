## Темы заданий:
1. Обзор методов.
2. Различия между процедурами и функциями.
3. Работа методов - примеры.
4. Управляющая структура return.
5. Использование сторожевых операторов (Guard Clauses).
6. Сторожевые операторы для защиты номинального варианта.
7. Использование ref и out аргументов в методах.
---
## Список задач

### 1. Обзор методов
1. Напишите метод, принимающий два целых числа и возвращающий их сумму.
2. Создайте метод, возвращающий максимальное из трех переданных чисел.
3. Реализуйте метод для определения, является ли число четным.
4. Напишите метод, выводящий все числа в диапазоне от `a` до `b`.
5. Опишите метод для расчета факториала числа.

### 2. Различия между процедурами (void-методами) и методами
1. Опишите метод, возвращающий квадрат числа.
2. Реализуйте метод, проверяющий, является ли строка палиндромом.
3. Создайте метод (процедуру), изменяющий значение переменной через `ref`.
4. Объясните различие между процедурой (методом, не возвращающим значение) и методом с возвращаемым значением на примере кода.
5. Реализуйте метод, который выводит строку `Hello, World!` на экран.

### 3. Работа методов — примеры
1. Реализуйте метод для поиска минимального элемента в массиве.
2. Реализуйте метод, принимающий массив и вычисляющий его среднее значение.
3. Опишите метод для копирования массива.
4. Сделайте метод, принимающий строку и возвращающий ее длину.
5. Создайте перегруженный метод для сложения двух и трех чисел.

### 4. Управляющая структура return
1. Создайте метод, возвращающий код ошибки, если входные данные некорректны.
2. Реализуйте метод, проверяющий делимость на 3 и возвращающий `true`/`false`.
3. Опишите метод, использующий `return` для досрочного выхода из метода до конца его выполнения.
4. Создайте метод, возвращающий `null`, если переданная строка пустая.
5. Реализуйте метод с несколькими `return`, которые завершают выполнение в зависимости от условий.

### 5. Использование сторожевых операторов (Guard Clauses)
1. Напишите метод с Guard Clause для проверки деления на ноль.
2. Реализуйте Guard Clause в конструкторе класса для проверки обязательного параметра.
3. Используйте Guard Clause для проверки диапазона допустимых значений.
4. Добавьте Guard Clause в метод: если массив пустой, немедленно возвращать управление (`return`).
5. Реализуйте метод с Guard Clause, выбрасывающим исключение при аргументе `null`.

### 6. Сторожевые операторы для защиты номинального варианта
1. Реализуйте Guard Clause для метода расчетов — если сумма отрицательная, вернуть ошибку.
2. В методе проверки возраста используйте Guard Clause — если возраст меньше минимума, выполните `return`.
3. Реализуйте Guard Clause для метода авторизации — если пароль слишком короткий, выбросить исключение.
4. Добавьте Guard Clause в метод регистрации, проверяющий уникальность email.
5. Сделайте Guard Clause, выбрасывающий исключение при невалидном формате даты.

### 7. Использование ref и out аргументов в методах
1. Реализуйте метод деления, возвращающий частное и остаток через `out`.
2. Опишите метод, заполняющий массив результатами через `out`.
3. Реализуйте метод, который возвращает результат выполнения (`true`/`false`) через `return`, а само вычисленное значение — через `out`.
4. Сделайте метод, увеличивающий значение аргумента через `ref`.
5. Создайте метод, меняющий местами два значения через `ref`.
